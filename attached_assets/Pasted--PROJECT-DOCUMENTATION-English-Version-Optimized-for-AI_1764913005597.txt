✅ PROJECT DOCUMENTATION (English Version – Optimized for AI Implementation)

System POS + Inventory + Cash Control + Multi-Store Management for Natural Products Stores (Python)

🏷️ 1. PROJECT TITLE

POS + Inventory + Cash Management System with Multi-Store Support for Natural Products Shops

🎯 2. GENERAL OBJECTIVE

Develop a professional Python-based application that manages multiple natural-products stores.
Each store must operate independently, with its own:

Inventory

Cash registers (cashier cash boxes)

Major cash box for physical cash

Sales history

Expenses

Transfer records

Store-specific financial reports

Additionally:

Global major accounts for Nequi, Daviplata, and Bold must be shared across all stores.

The system must include:

POS module

Multi-store inventory

User roles and permissions

Cash and account management

Transfers between stores

Auditing

Reports

Low-stock and expiration alerts

🧑‍💼 3. USER MODULE (ROLES & PERMISSIONS)
3.1 Roles
Administrator

Full access

Create and edit stores

Full inventory management

Create and edit users

Global and per-store reports

Manage all major and minor cash boxes

Create new payment methods

Supervisor

View reports and sales

View inventory

Cannot edit or delete inventory

Cannot access major cash boxes

Cannot create users

Cashier

Perform sales

Register expenses

Perform cash transfers between boxes

Open and close cash register

Cannot access full inventory

Cannot view global reports

📜 4. AUDIT LOG SYSTEM

Every action must be recorded with:

Username

Date and time

Store

Action description

Technical details

A dedicated audit review module is required.

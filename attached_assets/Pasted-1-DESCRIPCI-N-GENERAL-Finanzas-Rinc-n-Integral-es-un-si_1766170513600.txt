1. DESCRIPCIÓN GENERAL

Finanzas Rincón Integral es un sistema POS web diseñado para la gestión integral de tiendas físicas, enfocado en control de ventas, inventario, caja, egresos y análisis financiero real.

El sistema está pensado para:

Cajeros (operación diaria)

Administradores (control, análisis y toma de decisiones)

Actualmente gestiona dos tiendas:

20 de Julio

Tunal

2. OBJETIVO DEL SISTEMA

El objetivo del sistema no es solo vender, sino:

Medir ganancias reales

Analizar rendimiento del negocio

Detectar pérdidas, errores y oportunidades

Facilitar decisiones del administrador

Mantener una experiencia simple para el cajero

3. TECNOLOGÍAS (ESTADO ACTUAL)
Frontend

React 18 + TypeScript

Vite

Zustand (estado global con persistencia)

Tailwind CSS + Shadcn UI

Recharts (gráficas)

React Hook Form + Zod

Backend

Node.js + Express + TypeScript

Arquitectura basada en interfaces (IStorage)

MemStorage (temporal)

Backend preparado para migración futura a DB

⚠️ Nota importante
Actualmente los datos se manejan en cliente, pero el sistema está diseñado para migrar a persistencia real (SQLite / PostgreSQL) sin rehacer la app.

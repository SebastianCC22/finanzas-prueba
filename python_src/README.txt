# Finanzas Pro - Código Fuente Python

Este directorio contiene el código fuente completo de la aplicación de escritorio solicitada.

## Requisitos

1. Python 3.8+
2. Librerías:
   pip install PyQt5 matplotlib

## Estructura

- `main.py`: Archivo principal para ejecutar la app.
- `database/`: Lógica de base de datos SQLite.
- `ui/`: Ventanas e interfaz gráfica (PyQt5).

## Cómo Ejecutar

python main.py

## Cómo Compilar a .EXE

1. Instalar PyInstaller:
   pip install pyinstaller

2. Ejecutar el script:
   compilar.bat
